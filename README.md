# Yandex Music Controls

Extension to control Yandex Music tabs.

This extension provides you ability to control your Yandex.Music tabs.

### Features
  * Pause/Play, Next/Previous track
  * Change track repeat / shuffle
  * Like and Dislike
  * Hot keys
